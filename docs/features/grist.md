## What is Grist?

[Grist](https://www.getgrist.com/) is a modern relational spreadsheet that
combines the flexibility of spreadsheets with the power of databases. It
structures data like a database while maintaining a familiar spreadsheet
interface.

Grist supports Python formulas, custom layouts, forms, and granular access
control, making it ideal for teams needing flexible data management.

![Grist interface](../imgs/grist.png)

## Why Use Grist?

> Spreadsheet with database superpowers

**Key benefits:**

- **Database Structure**: Treats rows as records with relational capabilities
- **Python Formulas**: Full Python syntax support for advanced calculations
- **AI Assistant**: AI-powered formula generation using plain language
- **Custom Layouts**: Drag-and-drop widgets (tables, cards, charts)
- **Forms & Surveys**: Create and share forms with real-time data analysis
- **Access Control**: Granular permissions down to row and column level
- **SQLite Backend**: Portable, self-contained format based on SQLite
